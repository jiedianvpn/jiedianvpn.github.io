最新版Shadowrocket在功能层面实现了多维度的提升，不仅延续了以往的优势，更在协议支持、流量管理和用户体验上做出了重要突破。

1. 多协议兼容，满足多样化需求
新版支持Vmess、Shadowsocks、Trojan等主流代理协议，并针对各类协议进行了底层优化。无论是追求速度的游戏用户，还是注重隐私的办公群体，均可找到适合自己的配置方案。尤其值得一提的是，Trojan协议在新版本中表现突出，其伪装特性在严格网络环境中仍能保持稳定连接。

智能分流功能进一步强化，系统可根据网络延迟、服务器负载等因素动态选择最优节点。用户还可通过自定义规则实现精细化的流量控制，例如指定某类应用直连、某类网站走代理，甚至根据不同时间段切换节点策略。这一功能尤其适合跨境办公、学术研究等复杂场景。

3. 流量统计与连接可视化

4. 一键连接与快捷操作
简化操作流程是本次更新的另一大亮点。用户可通过小组件或快捷指令实现一键连接，甚至搭配自动化工具（如iOS快捷指令）实现定时开关代理，极大提升了使用便捷性。

  2. 全局与分应用代理：

    • 支持全局代理模式，也可指定特定应用（如Twitter、Netflix）使用代理，其他应用直连。

三、常见问题与解决方案

Q1: 连接频繁断开或速度不稳定？
- 尝试切换协议（如从Vmess改为Trojan）；
- 检查本地网络是否限制UDP传输；
- 切换至延迟更低的节点。

Q2: 如何备份配置与节点信息？
- 使用“配置导出”功能生成配置文件，保存至iCloud或本地文件；
- 建议定期导出订阅链接并加密存储。

Q3: 部分应用无法通过代理访问？
- 检查规则列表中是否遗漏该应用域名；
- 尝试关闭IPv6支持（在“设置”→“高级”中调整）。

Q4: 是否支持Android设备？
- Shadowrocket为iOS专属应用，Android用户可选用AnXray、V2rayNG等同类工具。

FAQ

Matsuri 支持哪些传输方式？
Matsuri 支持 TCP、gRPC、QUIC、WebSocket 等多种传输协议。用户可在配置文件中自由切换，根据服务器支持情况选择最优连接方式，以获得更低延迟与更强稳定性。
Nekoray 与 Nekobox 有什么不同？
Nekoray 是一款基于 Qt 的跨平台代理前端，而 Nekobox 更偏向移动端简化使用。两者均支持 V2Ray/VLESS/Trojan 协议，但 Nekobox 提供了更轻量的界面和更快的连接启动速度。
Matsuri 支持订阅更新吗？
支持。Matsuri 内置订阅更新模块，可自动检测链接变化并定期刷新节点。用户还可手动点击更新按钮同步最新配置，保持网络连接的可用性与速度。
Sing-box 是否兼容 Clash 配置文件？
部分兼容。Sing-box 支持从 Clash 订阅转换工具导入节点，但配置语法略有不同。建议使用第三方订阅转换器，如 sub-converter，将 Clash 格式转换为 Sing-box JSON 格式使用。
Matsuri 节点延迟排序如何提高访问速度？
Matsuri 定期测速节点延迟，结合策略组自动选择延迟最低节点处理流量。用户无需手动切换节点即可保持高速访问，提高网页浏览、视频播放和游戏体验的稳定性。
V2RayN 的作用是什么？
V2RayN 是 V2Ray 的 Windows 图形客户端，提供节点管理、订阅导入、日志查看等功能。它支持多协议配置，并可与系统代理集成，是 Windows 用户最常用的代理管理工具之一。
SSR 的协议插件与加密方式如何搭配使用？
协议插件可增强抗封锁能力，加密方式确保数据安全。合理搭配可模拟正常流量特征，同时保障传输安全，有助于在封锁严格环境下维持节点可用性和稳定连接。
Hysteria2 多路复用如何优化高频短连接？
多路复用允许多个短连接共享同一通道，减少握手次数和加密开销。适合游戏和即时通讯等高频短连接场景，提高吞吐量和连接稳定性，保证流量快速传输。
如何在 Nginx 上为 Vmess/WS 配置反向代理与 TLS？
在 Nginx 中配置 HTTPS 站点并将特定路径（如 /ray）反向代理到 V2Ray 的 WebSocket 端口，确保 proxy_set_header 设置正确（Host、X-Real-IP 等）。同时使用 certbot 获取并自动续期 TLS 证书以维护安全。
Surfboard 与 Shadowrocket 有什么区别？
两者都是 iOS 上的代理工具。Shadowrocket 功能更全面、更新更频繁，而 Surfboard 以轻量和快速见长。若只需简单分流与节点切换，Surfboard 足以满足日常使用。
